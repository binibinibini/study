# matplotlib.pyplot 모듈을 사용한 데이터 시각화
```python
# 1. Figure 객체 생성
fig = plt.figure()  # 스케치북 한장

# 2. Figure 객체에 Axis(축) 설정 - Subplot을 생성.(몇개의 그래프를 그릴거냐)
# axis = fig.add_subplot(1, 1, 1)  # add_subplot(nrows, ncols, index)
# 스케치북에 그림. 행의 갯수 1, 열의 갯수 2, 1개의 그래프를 그릴거

axis = fig.add_subplot(111)  # 간단하게 적을 수 있음. add_subplot(pos): '행/열/인덱스' 순서로 만들어진 3자리 정수

# 3. Axis에 그래프를 그림
axis.hist(x = tips.total_bill)

# 4. 그래프 옵션 설정
axis.set_title('Histogram(total_bill)')
axis.set_xlabel('Total Bill')   # x 레이블을 세팅
axis.set_ylabel('Count')

# 5. 그래프 보여주기
plt.show()  # 리턴값은 없음
```

# Histogram(히스토그램)
연속형 자료(숫자 타입 데이터)를 구간으로 나눠서 구간 안에 포함된 자료의 개수를 막대로 시각화한 그래프.

```python

plt.grid()
```

# Scatter Plot(삼점도)
```
- 두 변수(컬럼) 간의 상관 관계를 알고 싶을 때 사용.
- y ~ x 관계 시각화.
  - y: 관심 변수, 종속 변수.
  - x: 독립 변수.
```

```python
# tip ~ total_bill 상관 관계(tip이 y축, total_bill이 x축)
plt.scatter(x = tips.total_bill, y = tips.tip,      # x, y: x축, y축에 사용할 변수(컬럼)
            alpha = 0.3,        # alpha: 투명도(0= 투명, 1= 불투명). 0 ~ 1사이의 숫자를 주면 됨
            s = 20,     # s: size. 마커(점)의 크기.
            c = 'IndianRed',    # c: color. 마커(점)의 색깔.
            marker = '^')       # marker: 마커(점)의 모양(o, s, v, ^, *, +, ...).
plt.xlabel('Total bill')
plt.ylabel('Tip')
plt.grid()
plt.show()
```
![image](https://github.com/user-attachments/assets/fb8bb50b-ea2c-4b03-b1ad-511d3e0a23fb)
